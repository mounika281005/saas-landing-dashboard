# SaaS Landing + Dashboard

## Tech Stack
- Next.js
- Tailwind CSS
- LocalStorage Auth
- JSONPlaceholder API

## Features

### Landing Page
- Pixel-close Figma implementation
- Responsive design
- Hover + focus states

### Auth
- Fake login/signup
- Token stored in localStorage
- Protected routes

### Dashboard
- Summary stats
- Users list with:
  - Search
  - Sort
  - Pagination
  - User modal
  - Loading + error states

### Settings
- Profile form
- Theme toggle
- Persisted settings

## Setup

```bash
npm install
npm run dev
